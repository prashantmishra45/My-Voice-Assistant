# AMIII Roadmap

## v0.1 - Basic Voice Conversation

- Push-to-talk voice input.
- Speech-to-text transcription.
- LLM response.
- Text-to-speech output.
- Clear setup errors.

## v0.2 - LLM Provider Layer

- Stable Ollama adapter.
- Stable Groq adapter.
- User-selectable provider mode.
- Auto fallback mode.
- Provider tests and setup docs.

## v0.3 - Application Launcher

- Open configured applications.
- Close applications only after confirmation.
- Windows-first support.

## v0.4 - Browser Automation

- Open websites.
- Search the web.
- Navigate pages.
- Download files only after confirmation.

## v0.5 - Messaging Automation

- Draft WhatsApp, Instagram, Facebook, and LinkedIn messages.
- Always confirm before sending.
- Keep audit logs for sent messages.

## v0.6 - File Management

- Create, rename, move, and organize files.
- Require confirmation before deletion or risky moves.

## v0.7 - Coding Assistant

- Explain code.
- Generate and modify code.
- Run tests.
- Summarize Git changes.

## v0.8 - Memory System

- SQLite-backed preferences.
- Task history.
- Long-term memory retrieval.

## v0.9 - Multi-tool Agent

- Tool routing.
- Task planning.
- Multi-step execution with safety gates.

## v1.0 - Full Modular Assistant

- Stable modular assistant across voice, tools, memory, browser, files, coding, messaging, and safety.
- Documented setup and contribution workflow.
- Acceptance tests for all major capabilities.
